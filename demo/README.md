## Local Development

### Prerequisites

- Node.js >= 20
- [pnpm 9.12.3](https://pnpm.io/installation)

### Install dependencies

```bash
# cd ./demo
# install dependencies
# corepack enable
pnpm install
```

### Run

```bash
# run
pnpm dev
```
