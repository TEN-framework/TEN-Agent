## Local Development

### Prerequisites

- Node.js >= 20
- [Yarn >= 4](https://yarnpkg.com/getting-started)

### Install dependencies

```bash
# cd ./demo
# install dependencies
yarn install
```

### Run

```bash
# run
yarn dev
```
